---
title: 'Delete Collection'
api: 'DELETE /v1/collections/{readable_id}'
description: 'Permanently delete a collection and all associated data'
---

## Overview

Permanently delete a collection and all its data. This operation:

- Removes all synced data from the vector database
- Deletes all source connections within the collection
- Cancels any scheduled sync jobs
- Cleans up all related resources

<Warning>
This action cannot be undone. All data will be permanently deleted.
</Warning>

## Path Parameters

<ParamField path="readable_id" type="string" required>
  The unique readable identifier of the collection to delete
</ParamField>

## Response

Returns the deleted collection object.

<ResponseField name="id" type="string" required>
  UUID of the deleted collection
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the deleted collection
</ResponseField>

<ResponseField name="readable_id" type="string" required>
  Readable ID of the deleted collection
</ResponseField>

<ResponseField name="status" type="string" required>
  Final status of the collection
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X DELETE https://api.airweave.ai/v1/collections/finance-data-ab123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.airweave.ai/v1/collections/finance-data-ab123"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

response = requests.delete(url, headers=headers)
deleted_collection = response.json()
```

```typescript TypeScript
const url = 'https://api.airweave.ai/v1/collections/finance-data-ab123';

  'Authorization': 'Bearer YOUR_API_KEY'
};

```
