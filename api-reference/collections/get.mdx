---
title: 'Get Collection'
api: 'GET /v1/collections/{readable_id}'
description: 'Retrieve details of a specific collection'
---

## Overview

Retrieve complete details of a specific collection by its readable ID. Returns the collection configuration including sync settings, status, and metadata.

Use this to check the current state of a collection or to get configuration details before making updates.

## Path Parameters

<ParamField path="readable_id" type="string" required>
  The unique readable identifier of the collection (e.g., 'finance-data-ab123')
</ParamField>

## Response

Returns a single collection object.

<ResponseField name="id" type="string" required>
  Unique UUID identifier for the collection
</ResponseField>

<ResponseField name="name" type="string" required>
  Human-readable display name
</ResponseField>

<ResponseField name="readable_id" type="string" required>
  URL-safe unique identifier used in API endpoints
</ResponseField>

<ResponseField name="status" type="string" required>
  Current operational status: `NEEDS_SOURCE`, `ACTIVE`, or `ERROR`
</ResponseField>

<ResponseField name="vector_size" type="integer" required>
  Vector dimensions used by this collection
</ResponseField>

<ResponseField name="embedding_model_name" type="string" required>
  Name of the embedding model used
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  UUID of the organization that owns this collection
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp when created
</ResponseField>

<ResponseField name="modified_at" type="string" required>
  ISO 8601 timestamp when last modified
</ResponseField>

<ResponseField name="created_by_email" type="string">
  Email of the user who created this collection
</ResponseField>

<ResponseField name="modified_by_email" type="string">
  Email of the user who last modified this collection
</ResponseField>

<ResponseField name="sync_config" type="object">
  Default sync configuration for all syncs in this collection
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.airweave.ai/v1/collections/finance-data-ab123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.airweave.ai/v1/collections/finance-data-ab123"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

response = requests.get(url, headers=headers)
collection = response.json()
```

```typescript TypeScript
const url = 'https://api.airweave.ai/v1/collections/finance-data-ab123';
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY'
};

const response = await fetch(url, { headers });
const collection = await response.json();
```

</CodeGroup>
