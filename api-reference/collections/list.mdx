api: 'GET /v1/collections'
description: 'Retrieve all collections belonging to your organization'
---

## Overview

Retrieve all collections in your organization. Collections are containers that group related data from one or more source connections, enabling unified search across multiple data sources.

Results are sorted by creation date (newest first) and support pagination and text search filtering.

## Query Parameters

<ParamField query="skip" type="integer" default={0}>
  Number of collections to skip for pagination (minimum: 0)
</ParamField>

<ParamField query="limit" type="integer" default={100}>
  Maximum number of collections to return (1-1000)
</ParamField>

<ParamField query="search" type="string">
  Search term to filter collections by name or readable_id (e.g., "customer")
</ParamField>

## Response

Returns an array of collection objects.

<ResponseField name="id" type="string" required>
  Unique UUID identifier for the collection
</ResponseField>

<ResponseField name="name" type="string" required>
  Human-readable display name
</ResponseField>

<ResponseField name="readable_id" type="string" required>
  URL-safe unique identifier used in API endpoints
</ResponseField>

<ResponseField name="status" type="string" required>
  Current operational status: `NEEDS_SOURCE`, `ACTIVE`, or `ERROR`
</ResponseField>

<ResponseField name="vector_size" type="integer" required>
  Vector dimensions used by this collection
</ResponseField>

<ResponseField name="embedding_model_name" type="string" required>
  Name of the embedding model used
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  UUID of the organization that owns this collection
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp when created
</ResponseField>

<ResponseField name="modified_at" type="string" required>
  ISO 8601 timestamp when last modified
</ResponseField>

<ResponseField name="created_by_email" type="string">
  Email of the user who created this collection
</ResponseField>

<ResponseField name="modified_by_email" type="string">
  Email of the user who last modified this collection
</ResponseField>

<ResponseField name="sync_config" type="object">
  Default sync configuration for all syncs in this collection
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.airweave.ai/v1/collections?limit=10 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.airweave.ai/v1/collections"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

collections = response.json()
```

## Status Values

- **NEEDS_SOURCE**: Collection has no authenticated connections, or connections exist but haven't synced yet
- **ACTIVE**: At least one connection has completed a sync or is currently syncing
- **ERROR**: All connections have failed their last sync
