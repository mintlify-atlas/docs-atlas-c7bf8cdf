---
title: 'Update Collection'
api: 'PATCH /v1/collections/{readable_id}'
description: "Update an existing collection's properties"
---

## Overview

Update a collection's display name or sync configuration. The `readable_id` cannot be changed after creation to maintain stable API endpoints.

## Path Parameters

<ParamField path="readable_id" type="string" required>
  The unique readable identifier of the collection to update
</ParamField>

## Request Body

All fields are optional. Only include fields you want to change.

<ParamField body="name" type="string">
  Updated display name for the collection (4-64 characters)
</ParamField>

<ParamField body="sync_config" type="object">
  Default sync configuration for all syncs in this collection
  
  <Expandable title="properties">
    <ParamField body="handlers" type="object">
      <ParamField body="enable_vector_handlers" type="boolean">
        Enable vector database handlers
      </ParamField>
      <ParamField body="enable_postgres_handler" type="boolean">
        Enable PostgreSQL metadata handlers
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

## Response

Returns the updated collection object.

<ResponseField name="id" type="string" required>
  Unique UUID identifier for the collection
</ResponseField>

<ResponseField name="name" type="string" required>
  Updated display name
</ResponseField>

<ResponseField name="readable_id" type="string" required>
  URL-safe identifier (unchanged)
</ResponseField>

<ResponseField name="modified_at" type="string" required>
  ISO 8601 timestamp when last modified
</ResponseField>

<ResponseField name="modified_by_email" type="string">
  Email of the user who made this update
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X PATCH https://api.airweave.ai/v1/collections/finance-data-ab123 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Updated Finance Data"}'
```

```python Python
import requests

url = "https://api.airweave.ai/v1/collections/finance-data-ab123"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {"name": "Updated Finance Data"}

response = requests.patch(url, headers=headers, json=data)
collection = response.json()
```

```typescript TypeScript
const url = 'https://api.airweave.ai/v1/collections/finance-data-ab123';
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
};
const data = { name: 'Updated Finance Data' };

const response = await fetch(url, {
  method: 'PATCH',
  headers: headers,
  body: JSON.stringify(data)
});
const collection = await response.json();
```

</CodeGroup>
