---
title: 'Get Sync Job'
api: 'GET /v1/sync/jobs/{job_id}'
description: 'Retrieve details of a specific sync job'
---

## Overview

Get detailed information about a specific sync job, including status, timing, and entity processing statistics.

## Path Parameters

<ParamField path="job_id" type="string" required>
  Unique UUID identifier of the sync job
  
  Example: "770e8400-e29b-41d4-a716-446655440002"
</ParamField>

## Response

<ResponseField name="id" type="string" required>
  Unique UUID identifier of the sync job
</ResponseField>

<ResponseField name="sync_id" type="string" required>
  UUID of the sync configuration this job belongs to
</ResponseField>

<ResponseField name="sync_name" type="string">
  Name of the sync (populated from join query)
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  UUID of the organization
</ResponseField>

<ResponseField name="status" type="string" required>
  Current job status: `created`, `pending`, `in_progress`, `completed`, `failed`, or `cancelled`
</ResponseField>

<ResponseField name="scheduled" type="boolean" required>
  Whether this job was triggered by a schedule (`true`) or manually (`false`)
</ResponseField>

<ResponseField name="entities_inserted" type="integer">
  Number of new entities created
</ResponseField>

<ResponseField name="entities_updated" type="integer">
  Number of existing entities updated
</ResponseField>

<ResponseField name="entities_deleted" type="integer">
  Number of entities removed
</ResponseField>

<ResponseField name="entities_kept" type="integer">
  Number of entities that were checked but required no changes
</ResponseField>

<ResponseField name="entities_skipped" type="integer">
  Number of entities skipped due to filtering rules
</ResponseField>

<ResponseField name="entities_encountered" type="object">
  Detailed breakdown of entities processed by type or category
</ResponseField>

<ResponseField name="started_at" type="string">
  When the job began active processing (ISO 8601)
</ResponseField>

<ResponseField name="completed_at" type="string">
  When the job finished successfully (ISO 8601)
</ResponseField>

<ResponseField name="failed_at" type="string">
  When the job failed (ISO 8601)
</ResponseField>

<ResponseField name="error" type="string">
  Detailed error message if the job failed
</ResponseField>

<ResponseField name="sync_config" type="object">
  Sync configuration used for this job
</ResponseField>

<ResponseField name="sync_metadata" type="object">
  Additional metadata about the sync
</ResponseField>

<ResponseField name="created_at" type="string">
  When the job was created (ISO 8601)
</ResponseField>

<ResponseField name="modified_at" type="string">
  When the job was last modified (ISO 8601)
</ResponseField>

<ResponseField name="created_by_email" type="string">
  Email of the user who created the job (for manually triggered jobs)
</ResponseField>

<ResponseField name="modified_by_email" type="string">
  Email of the user who last modified the job
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl "https://api.airweave.ai/v1/sync/jobs/770e8400-e29b-41d4-a716-446655440002" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

job_id = "770e8400-e29b-41d4-a716-446655440002"
url = f"https://api.airweave.ai/v1/sync/jobs/{job_id}"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

response = requests.get(url, headers=headers)
job = response.json()

print(f"Job: {job['sync_name']}")
print(f"Status: {job['status']}")
print(f"Entities: +{job['entities_inserted']} ~{job['entities_updated']} -{job['entities_deleted']}")

if job['completed_at'] and job['started_at']:
    from datetime import datetime
    start = datetime.fromisoformat(job['started_at'].replace('Z', '+00:00'))
    end = datetime.fromisoformat(job['completed_at'].replace('Z', '+00:00'))
    duration = (end - start).total_seconds()
    print(f"Duration: {duration:.1f} seconds")
```

```typescript TypeScript
const jobId = '770e8400-e29b-41d4-a716-446655440002';
const url = `https://api.airweave.ai/v1/sync/jobs/${jobId}`;
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY'
};

const response = await fetch(url, { headers });
const job = await response.json();

console.log(`Job: ${job.sync_name}`);
console.log(`Status: ${job.status}`);
console.log(`Entities: +${job.entities_inserted} ~${job.entities_updated} -${job.entities_deleted}`);

if (job.completed_at && job.started_at) {
  const start = new Date(job.started_at);
  const end = new Date(job.completed_at);
  const duration = (end.getTime() - start.getTime()) / 1000;
  console.log(`Duration: ${duration.toFixed(1)} seconds`);
}
```
