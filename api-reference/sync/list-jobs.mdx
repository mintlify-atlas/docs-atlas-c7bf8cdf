## Overview

Retrieve all sync jobs across all syncs in your organization. This provides a unified view of data synchronization activity.

**Use Cases:**
- Monitor all sync activity across your organization
- Track sync performance and reliability
- Identify failed syncs that need attention
- Audit data refresh operations

## Query Parameters

<ParamField query="skip" type="integer" default={0}>
  Number of jobs to skip for pagination
</ParamField>

<ParamField query="limit" type="integer" default={100}>
  Maximum number of jobs to return (1-100)
</ParamField>

<ParamField query="status" type="array">
  Filter by job status. Can specify multiple statuses.
  
  Valid values: `created`, `pending`, `in_progress`, `completed`, `failed`, `cancelled`
  
  Example: `?status=failed&status=in_progress`
</ParamField>

## Response

Returns an array of sync job objects.

<ResponseField name="id" type="string" required>
  Unique UUID identifier of the sync job
</ResponseField>

<ResponseField name="sync_id" type="string" required>
  UUID of the sync configuration this job belongs to
</ResponseField>

<ResponseField name="sync_name" type="string">
  Name of the sync (populated from join query)
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  UUID of the organization
</ResponseField>

<ResponseField name="status" type="string" required>
  Current job status:
  - `created`: Job created but not queued
  - `pending`: Queued and waiting to start
  - `in_progress`: Currently running
  - `completed`: Finished successfully
  - `failed`: Encountered errors
  - `cancelled`: Manually cancelled
</ResponseField>

<ResponseField name="scheduled" type="boolean" required>
  Whether this job was triggered by a schedule (`true`) or manually (`false`)
</ResponseField>

<ResponseField name="entities_inserted" type="integer">
  Number of new entities created
</ResponseField>

<ResponseField name="entities_updated" type="integer">
  Number of existing entities updated
</ResponseField>

<ResponseField name="entities_deleted" type="integer">
  Number of entities removed
</ResponseField>

<ResponseField name="entities_kept" type="integer">
  Number of entities checked but unchanged
</ResponseField>

<ResponseField name="entities_skipped" type="integer">
  Number of entities skipped due to filters or rules
</ResponseField>

<ResponseField name="entities_encountered" type="object">
  Breakdown of entities processed by type
</ResponseField>

<ResponseField name="started_at" type="string">
  When the job began processing (ISO 8601)
</ResponseField>

<ResponseField name="completed_at" type="string">
  When the job finished (ISO 8601)
</ResponseField>

<ResponseField name="failed_at" type="string">
  When the job failed (ISO 8601)
</ResponseField>

<ResponseField name="error" type="string">
  Error message if the job failed
</ResponseField>

<ResponseField name="created_at" type="string">
  When the job was created (ISO 8601)
</ResponseField>

<ResponseField name="modified_at" type="string">
  When the job was last modified (ISO 8601)
</ResponseField>

<ResponseField name="created_by_email" type="string">
  Email of the user who created the job
</ResponseField>

<ResponseField name="modified_by_email" type="string">
  Email of the user who last modified the job
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl "https://api.airweave.ai/v1/sync/jobs?status=failed&limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.airweave.ai/v1/sync/jobs"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
params = {
    "status": ["failed", "in_progress"],
    "limit": 20
}

response = requests.get(url, headers=headers, params=params)
jobs = response.json()

for job in jobs:
    print(f"Job {job['id']}: {job['sync_name']} - {job['status']}")
    if job['status'] == 'failed':
        print(f"  Error: {job['error']}")
```

```typescript TypeScript
const params = new URLSearchParams();
params.append('status', 'failed');
params.append('status', 'in_progress');
params.append('limit', '20');

const url = `https://api.airweave.ai/v1/sync/jobs?${params}`;
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY'
};

const response = await fetch(url, { headers });
const jobs = await response.json();

jobs.forEach(job => {
  console.log(`Job ${job.id}: ${job.sync_name} - ${job.status}`);
  if (job.status === 'failed') {
    console.log(`  Error: ${job.error}`);
  }
});
```
