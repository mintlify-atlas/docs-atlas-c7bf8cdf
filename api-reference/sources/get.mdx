---
title: 'Get Source'
api: 'GET /v1/sources/{short_name}'
description: 'Retrieve detailed information about a specific data source connector'
---

## Overview

Retrieve the complete configuration for a source type, including:

- **Authentication fields**: Schema for credentials required to connect
- **Configuration fields**: Schema for optional settings and customization
- **Supported auth providers**: Pre-configured OAuth providers available for this source

Use this endpoint before creating a source connection to understand what authentication and configuration values are required.

## Path Parameters

<ParamField path="short_name" type="string" required>
  Technical identifier of the source type (e.g., 'github', 'stripe', 'slack')
</ParamField>

## Response

<ResponseField name="name" type="string" required>
  Human-readable name of the data source
</ResponseField>

<ResponseField name="short_name" type="string" required>
  Technical identifier
</ResponseField>

<ResponseField name="description" type="string">
  Detailed description of what data this source can extract
</ResponseField>

<ResponseField name="auth_methods" type="array">
  List of supported authentication methods
</ResponseField>

<ResponseField name="oauth_type" type="string">
  OAuth token type for OAuth sources
</ResponseField>

<ResponseField name="auth_fields" type="object">
  Schema definition for authentication fields (only for direct auth)
</ResponseField>

<ResponseField name="config_fields" type="object" required>
  Schema definition for configuration fields
</ResponseField>

<ResponseField name="supported_auth_providers" type="array">
  List of auth provider short names that support this source
</ResponseField>

<ResponseField name="labels" type="array">
  Categorization tags
</ResponseField>

<ResponseField name="supports_continuous" type="boolean" required>
  Whether this source supports cursor-based continuous syncing
</ResponseField>

<ResponseField name="federated_search" type="boolean" required>
  Whether this source uses federated search
</ResponseField>

<ResponseField name="supports_access_control" type="boolean" required>
  Whether this source supports document-level access control
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.airweave.ai/v1/sources/github \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.airweave.ai/v1/sources/github"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

response = requests.get(url, headers=headers)
source = response.json()

print(f"Source: {source['name']}")
print(f"Auth methods: {source['auth_methods']}")
```

```typescript TypeScript
const url = 'https://api.airweave.ai/v1/sources/github';
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY'
};

const response = await fetch(url, { headers });
const source = await response.json();

console.log(`Source: ${source.name}`);
console.log(`Auth methods: ${source.auth_methods}`);
```

</CodeGroup>
